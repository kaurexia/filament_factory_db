# 🏭 Filament Factory DB - Управление производством филамента

Учебное приложение с графическим интерфейсом для управления производством пластикового филамента. Приложение предоставляет полный контроль над производственной базой данных через интуитивный интерфейс на русском языке.

![Python](https://img.shields.io/badge/Python-3.8%2B-blue)
![MySQL](https://img.shields.io/badge/MySQL-8.0%2B-orange)
![Tkinter](https://img.shields.io/badge/GUI-Tkinter-green)
![Лицензия](https://img.shields.io/badge/Лицензия-MIT-yellow)

## 🎯 Обзор
Система поддерживает полный цикл производства:
- 📦 Учёт сырья и материалов
- 🏭 Управление производственными процессами
- 🎨 Контроль цветов и диаметров филамента
- 📊 Контроль качества продукции
- 🏪 Управление складскими запасами
- 👥 Учёт сотрудников и оборудования

## 🛠 Технологии
- **Python 3.8+** — основной язык программирования
- **Tkinter** — графический интерфейс пользователя
- **MySQL 8.0+** — система управления базами данных
- **mysql-connector-python** — драйвер для подключения к MySQL
- **Хранимые процедуры и функции** — бизнес-логика на уровне БД
- **Триггеры** — автоматизация операций с данными

## ✨ Функциональность

### 📊 Управление таблицами (18 таблиц)
- **Сотрудники** — персонал и должности
- **Оборудование** — станки и их состояние
- **Сырьё и материалы** — типы и поставки
- **Производственные процессы** — этапы изготовления
- **Продукция** — готовый филамент
- **Контроль качества** — тестирование продукции
- **Склады** — хранение сырья и продукции
- **Отчеты** — автоматическая генерация
- **Пользователи** — система авторизации
- **Поставщики** — партнеры по сырью

### ⚙ Операции с данными
- **CRUD операции** — полный набор для всех таблиц
- **Поиск** — глобальный поиск по всем таблицам
- **Сортировка** — по любому столбцу в обоих направлениях
- **JOIN операций** — визуальное объединение таблиц
- **Внешние ключи** — интеллектуальные выпадающие списки

### 🔧 Хранимые процедуры и функции
- **pack_product** — упаковка готовой продукции
- **add_material** — добавление нового сырья
- **start_process** — запуск производственного процесса
- **finish_process** — завершение процесса производства
- **quality_passed** — проверка качества продукции
- **`rawmaterial_used_kg`** — расчет расхода сырья
- **`days_until_rawmaterial_runs_out`** — прогноз остатка сырья

### 🛡 Валидация и безопасность
- Проверка типов данных (int, float, date, datetime)
- Валидация форматов дат и времени
- Проверка неотрицательных значений
- Защита от удаления связанных записей (FK constraints)
- Обработка ошибок с понятными сообщениями

## 🚀 Установка и запуск

### Предварительные требования
1. Установите **Python 3.8** или новее
2. Установите **MySQL Server 8.0** или новее
3. Создайте базу данных для проекта

### Шаг 1: Клонирование репозитория

git clone https://github.com/ваш-username/filament-factory.git
cd filament-factory

### Шаг 2: Установка зависимостей

pip install mysql-connector-python

### Шаг 3: Настройка базы данных
1. Создайте базу данных:

CREATE DATABASE filament_factory;
USE filament_factory;

2. Выполните SQL-скрипт создания структуры (предоставляется отдельно)

### Шаг 4: Запуск приложения

python mysql_gui.py

### Параметры подключения по умолчанию:

ХОСТ: localhost
ПОЛЬЗОВАТЕЛЬ: root
ПАРОЛЬ: root312
БАЗА ДАННЫХ: filament_factory

## 🖥 Использование

### 1. Авторизация
- Введите параметры подключения к MySQL
- Используйте тестовые данные или свои реквизиты

### 2. Работа с таблицами
- Выберите таблицу из списка
- Используйте кнопки для добавления/редактирования/удаления
- Используйте поиск для фильтрации данных
- Сортируйте данные по нужным столбцам

### 3. Выполнение процедур
- Откройте раздел "Процедуры и функции"
- Выберите нужную процедуру
- Заполните параметры в форме
- Просмотрите результаты выполнения

### 4. Объединение таблиц (JOIN)
- Используйте окно "Объединить таблицы"
- Выберите две таблицы для объединения
- Укажите столбцы для связи
- Выберите тип JOIN (INNER, LEFT, RIGHT)

### 5. Поиск по всей базе
- Введите поисковый запрос
- Выберите "Все таблицы" для глобального поиска
- Просмотрите результаты во вкладках
